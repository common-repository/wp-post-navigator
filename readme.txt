Adds the Previous and Next Post Navigator Buttons on the left and right side of your WordPress Post.

== Description ==

WP Post Navigator is a WordPress Plugin to Add Previous/Next Post Navigation Button. It works automatically once the Plugin is activated.

= WP Post Navigator Features =

 1. It works automatically when the Plugin is activated.
 2. It has an option ON/OFF to disable the Previous/Next Buttons.
 3. 5 different types of Button styles to choose from Settings Page.
 4. Should work in all major Browsers and WordPress themes.
 5. Simple and Easy to set up.

== Installation ==

1. Upload `wp-post-navigator` Folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings-> WP Post Navigator
